Reparent Handler Demo

This demo shows how reparenting of nodes can be customized.

This is done with a custom implementation of the interface IReparentNodeHandler that is set as default reparent handler of the input mode of the graph component.

Things to Try

Select and move a node. While dragging, press the Shift key to add or remove a node from a group. Angled indicators are shown at the group's corners if adding or removing is possible.

Node Types

  • Red nodes cannot be added to any group.
  • Green nodes and blue nodes can be added only to groups of the same color.
  • Green nodes can be reparented without pressing the Shift key.
