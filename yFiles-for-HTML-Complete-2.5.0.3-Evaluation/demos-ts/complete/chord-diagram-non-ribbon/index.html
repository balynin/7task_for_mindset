Non-ribbon Chord Diagram

This demo shows how to create a non-ribbon chord diagram. In contrast to normal chord diagrams, these diagrams use clearly separated nodes and edges, so it can be easier to see the individual connections.

This demo illustrates email communication between employees of a fictitious company.

The layout of the graph is calculated with the circular layout. The color of the nodes represents the department to which each employee belongs. Each edge has the two colors of its source and target. The thickness of the edge is proportional to the number of e-mails exchanged between the two nodes.
