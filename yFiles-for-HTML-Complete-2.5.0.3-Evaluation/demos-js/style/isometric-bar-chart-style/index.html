Isometric Bar Chart Node Style

Isometric Bar-Chart Augmentation

This demo shows how a diagram can be augmented with isometric bars providing additional information about the nodes.

In order to achieve this, the style from the isometric drawing demo is reused and rendered as an additional visualization for nodes. Similarly, another node style is put on top to render the label. These styles are added using their own GraphModelManager

Things to Try

  • Try the other options for Augmented Information to see the bars adjust to the provided information.
  • Toggle label visibility in the toolbar
  • Change the Augmented Information to None to see the original graph without the augmentations.
